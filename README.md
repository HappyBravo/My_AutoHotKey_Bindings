# My_AutoHotKey_Bindings

```FYI: 
# = Win
+ = Shift
! = Alt
^ = Ctrl
{Left} = Left Arrow Key
{Right} = Right Arrow Key
WheelUp = Scroll Wheel Up
WheelDown = Scroll Wheel down
```
